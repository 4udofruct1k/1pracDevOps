Redis Docker Container

Описание

Этот Docker-контейнер предназначен для развёртывания in-memory базы данных Redis, используемой как кеш, хранилище структур данных или брокер сообщений.

Назначение: обеспечить простую и настраиваемую среду для запуска Redis в контейнере.

Поддерживаемые аргументы образа
Аргумент	Описание
REDIS_VERSION	Версия Redis, устанавливаемая при сборке.
REDIS_PORT	Порт Redis внутри контейнера (по умолчанию 6379).
REDIS_USER	Пользователь, под которым запускается Redis.
REDIS_CONFIG_FILE	Путь к используемому конфигурационному файлу.
Переменные окружения
Переменная	Описание	Пример
REDIS_PASSWORD	Пароль для авторизации в Redis.	secret123
REDIS_APPENDONLY	Включение режима AOF (yes / no).	yes
REDIS_MAXMEMORY	Ограничение памяти.	256mb
REDIS_BIND	Адрес для привязки Redis.	0.0.0.0
Пример запуска контейнера
docker run -d \
  --name my-redis \
  -p 6379:6379 \
  -e REDIS_PASSWORD=supersecret \
  my-redis-image:latest

Настройка через переменные окружения
Основные настройки:

REDIS_PASSWORD — устанавливает пароль.

REDIS_APPENDONLY — включает AOF-персистентность.

REDIS_MAXMEMORY — задаёт максимальный объём памяти.

REDIS_BIND — изменяет адрес привязки.

Примеры значений:

REDIS_APPENDONLY=yes

REDIS_MAXMEMORY=512mb

REDIS_BIND=0.0.0.0
